تعلم Python

هنا ستتعلم أساسيات لغة Python خطوة بخطوة.

ما هي Python؟

Python هي لغة برمجة سهلة التعلم وقوية، تستخدم في تطوير الويب، تحليل البيانات، الذكاء الاصطناعي، وغيرها. Python كتكون لغة بسيطة وسهلة الفهم، وكتعتمد على كتابة الكود بطريقة نظيفة ومنظمة.

المتغيرات (Variables)

المتغيرات هي أماكن فالميموار لي كتخزن البيانات. ف Python، كتقدر تخزن أي نوع ديال البيانات (أرقام، نصوص، قوائم، إلخ) فمتغير. 

            
# مثال على المتغيرات
اسم = "أحمد"
عمر = 25
طول = 1.75

الجمل الشرطية (If-Else)

الجمل الشرطية كتخلي البرنامج يقرر شحال حاجة على أساس شرط معين. مثال: إيلا كان العمر أكبر من 18، نطبع "أنت بالغ"، وإلا نطبع "أنت قاصر". 

            
# مثال على الجمل الشرطية
عمر = 20

if عمر > 18:
    print("أنت بالغ")
else:
    print("أنت قاصر")

الحلقات التكرارية (Loops)

الحلقات التكرارية كتخلي البرنامج يعيد شحال حاجة. ف Python، كاينين نوعين ديال الحلقات: `for` و `while`. 

            
# مثال على حلقة for
for رقم in range(5):
    print("الرقم:", رقم)

# مثال على حلقة while
عمر = 0
while عمر < 5:
    print("العمر:", عمر)
    عمر += 1

الدوال (Functions)

الدوال هي وحدات من الكود لي كتقوم بمهمة معينة. يمكن استدعاء الدوال من مختلف أجزاء البرنامج. 

            
# مثال على دالة
def سلم_عليك(اسم):
    print("السلام عليكم،", اسم)

سلم_عليك("أحمد")

القوائم (Lists)

القوائم هي هياكل بيانات لتخزين مجموعة من العناصر من نفس النوع أو أنواع مختلفة. 

            
# مثال على القوائم
أسماء = ["أحمد", "محمد", "علي"]
أعمار = [25, 30, 22]

# طباعة العناصر
for اسم in أسماء:
    print(اسم)

القواميس (Dictionaries)

القواميس هي هياكل بيانات لتخزين البيانات بشكل أزواج مفتاح-قيمة. 

            
# مثال على القواميس
معلومات = {
    "اسم": "أحمد",
    "عمر": 25,
    "مهنة": "مبرمج"
}

print("الاسم:", معلومات["اسم"])
print("العمر:", معلومات["عمر"])

المزيد من النصائح

- استمر في التعلم وكن دائمًا على اطلاع على التقنيات الجديدة.
- شارك في المشاريع المفتوحة المصدر لاكتساب المزيد من الخبرة.
- اتبع الدروس والموارد المتاحة عبر الإنترنت لمواكبة التطورات.
