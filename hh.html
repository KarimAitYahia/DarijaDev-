تعلم تطوير المواقع

دروس HTML

1. مقدمة عن HTML

HTML هي لغة ترميز تستخدم لإنشاء هيكل صفحات الويب.

<!DOCTYPE html>
<html>
<head>
    <title>عنوان الصفحة</title>
</head>
<body>
    <h1>مرحبًا بالعالم!</h1>
    <p>هذا هو أول موقع لي.</p>
</body>
</html>

2. العناوين والفقرات

استخدم العناوين من <h1> إلى <h6> والفقرات <p>.

<h1>عنوان رئيسي</h1>
<h2>عنوان فرعي</h2>
<p>هذا نص فقرة.</p>

3. الروابط والصور

استخدم <a> للروابط و<img> للصور.

<a href="https://www.example.com">زيارة الموقع</a>
<img src="image.jpg" alt="وصف الصورة">

4. الجداول

استخدم <table> لإنشاء جداول.

<table>
    <tr>
        <th>الاسم</th>
        <th>العمر</th>
    </tr>
    <tr>
        <td>أحمد</td>
        <td>25</td>
    </tr>
    <tr>
        <td>محمد</td>
        <td>30</td>
    </tr>
</table>

5. النماذج

استخدم <form> لإنشاء نماذج تفاعلية.

<form>
    <label for="name">الاسم:</label>
    <input type="text" id="name" name="name">
    <br>
    <label for="email">البريد الإلكتروني:</label>
    <input type="email" id="email" name="email">
    <br>
    <input type="submit" value="إرسال">
</form>

دروس CSS

1. مقدمة عن CSS

CSS تستخدم لتنسيق عناصر HTML.

body {
    background-color: #f4f4f4;
    font-family: Arial, sans-serif;
}

2. تنسيق النصوص

استخدم CSS لتنسيق النصوص.

h1 {
    color: #00ff88;
    font-size: 2.5rem;
    text-align: center;
}

p {
    color: #333;
    font-size: 1.2rem;
    line-height: 1.6;
}

3. تنسيق العناصر

استخدم CSS لتنسيق العناصر مثل الأزرار والقوائم.

button {
    background-color: #7000ff;
    color: white;
    padding: 10px 20px;
    border: none;
    border-radius: 5px;
    cursor: pointer;
}

button:hover {
    background-color: #00ff88;
}

دروس JavaScript

1. مقدمة عن JavaScript

JavaScript تستخدم لإضافة تفاعل إلى صفحات الويب.

function sayHello() {
    alert("مرحبًا بالعالم!");
}

2. التعامل مع الأحداث

استخدم JavaScript للتعامل مع أحداث المستخدم.

document.getElementById('myButton').addEventListener('click', function() {
    alert('تم النقر على الزر!');
});

3. التعديل على DOM

استخدم JavaScript لتعديل عناصر الصفحة.

document.getElementById('myText').innerHTML = 'تم تغيير النص!';

مشاريع صغيرة

1. مشروع: آلة حاسبة بسيطة

قم بإنشاء آلة حاسبة بسيطة باستخدام HTML وCSS وJavaScript.

<div id="calculator">
    <input type="text" id="display" disabled>
    <div>
        <button onclick="appendNumber('1')">1</button>
        <button onclick="appendNumber('2')">2</button>
        <button onclick="appendNumber('3')">3</button>
        <button onclick="appendOperator('+')">+</button>
    </div>
    <div>
        <button onclick="calculate()">=</button>
        <button onclick="clearDisplay()">C</button>
    </div>
</div>

<script>
    let display = document.getElementById('display');

    function appendNumber(number) {
        display.value += number;
    }

    function appendOperator(operator) {
        display.value += operator;
    }

    function calculate() {
        display.value = eval(display.value);
    }

    function clearDisplay() {
        display.value = '';
    }
</script>
