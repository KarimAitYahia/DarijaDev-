مقدمة على الجافا

الجافا هي لغة برمجة قوية وسهلة التعلم. كتستعمل فمجالات بزاف، منها تطوير التطبيقات، الويب، والألعاب. فهاد الصفحة، غاندوزو مع بعض خطوة بخطوة باش نتعلمو الجافا من الصفر.

الأساسيات ديال الجافا

قبل ما نبداو نبرمجو، لازم نفهمو الأساسيات ديال الجافا. هاد المفاهيم هي اللبنات الأولى لي غادي نستعملوهم فكل شي غادي نتعلموه.

  • المتغيرات (Variables): هي أماكن فالميموار لي كتخزن البيانات. مثال: int age = 25;
  • أنواع البيانات (Data Types): كل متغير عندو نوع ديالو. مثال: int (عدد صحيح)، String (نص).
  • الجمل الشرطية (If-Else): كتخلي البرنامج يقرر شحال حاجة على أساس شرط معين. مثال: 
                      
if (age > 18) {
    System.out.println("أنت بالغ");
} else {
    System.out.println("أنت قاصر");
}
  • الحلقات التكرارية (Loops): كتخلي البرنامج يعيد شحال حاجة. مثال: 
                      
for (int i = 0; i < 5; i++) {
    System.out.println("الرقم: " + i);
}

الدوال (Functions)

الدوال هي وحدات من الكود لي كتقوم بمهمة معينة. يمكن استدعاء الدوال من مختلف أجزاء البرنامج. مثال: 

          
public class Main {
    public static void main(String[] args) {
        sayHello();
    }

    public static void sayHello() {
        System.out.println("السلام عليكم!");
    }
}

البرمجة الموازية (Concurrency)

البرمجة الموازية تسمح بتنفيذ عدة أوامر في وقت واحد، مما يحسن أداء البرنامج. مثال: 

          
public class Main {
    public static void main(String[] args) {
        Thread thread1 = new Thread(() -> {
            System.out.println("العملية الأولى");
        });

        Thread thread2 = new Thread(() -> {
            System.out.println("العملية الثانية");
        });

        thread1.start();
        thread2.start();
    }
}

المدخلات والمخرجات (Input/Output)

التعامل مع المدخلات والمخرجات هو جزء أساسي من البرمجة. مثال بسيط لقراءة وكتابة الملفات: 

          
import java.io.BufferedReader;
import java.io.FileReader;
import java.io.FileWriter;
import java.io.IOException;

public class FileIO {
    public static void main(String[] args) {
        // كتابة إلى ملف
        try (FileWriter writer = new FileWriter("example.txt")) {
            writer.write("السلام عليكم\n");
        } catch (IOException e) {
            e.printStackTrace();
        }

        // قراءة من ملف
        try (BufferedReader reader = new BufferedReader(new FileReader("example.txt"))) {
            String line;
            while ((line = reader.readLine()) != null) {
                System.out.println(line);
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

برمجة الكائنات (OOP)

الجافا كتستعمل مفهوم البرمجة الكائنية (OOP). هاد المفهوم كيعتمد على "الكائنات" لي هي عبارة عن بيانات ووظائف مرتبطين ببعض.

  • الفئات (Classes): هي القوالب لي كتستعمل لإنشاء الكائنات. مثال: 
                      
class Person {
    String name;
    int age;

    void sayHello() {
        System.out.println("سلام، أنا " + name);
    }
}
  • الكائنات (Objects): هي النسخ لي كتتخلق من الفئات. مثال: 
                      
Person person1 = new Person();
person1.name = "أحمد";
person1.age = 25;
person1.sayHello();
  • الوراثة (Inheritance): كتخلي فئة ترث من فئة أخرى. مثال: 
                      
class Student extends Person {
    String school;

    void study() {
        System.out.println(name + " كيدرس ف " + school);
    }
}

المصفوفات (Arrays)

المصفوفات هي هياكل بيانات لتخزين مجموعة من العناصر من نفس النوع. مثال: 

          
int[] numbers = {1, 2, 3, 4, 5};
for (int number : numbers) {
    System.out.println(number);
}

الاستثناءات (Exceptions)

الاستثناءات هي أخطاء تحدث أثناء تنفيذ البرنامج. يمكن التعامل معها باستخدام `try-catch`. مثال: 

          
try {
    int result = 10 / 0;
} catch (ArithmeticException e) {
    System.out.println("خطأ: القسمة على صفر!");
}

المجموعات (Collections)

المجموعات هي هياكل بيانات متقدمة لتخزين ومعالجة البيانات. مثال: 

          
import java.util.ArrayList;

ArrayList names = new ArrayList<>();
names.add("أحمد");
names.add("محمد");
names.add("علي");

for (String name : names) {
    System.out.println(name);
}

تطوير تطبيقات Android

الجافا كتستعمل بشكل كبير فتطوير تطبيقات Android. هنا غاندوزو كيفاش نبداو نطورو تطبيقات بسيطة باستخدام Android Studio.

  • إعداد بيئة العمل: نزل Android Studio وكون جاهز باش تبدا.
  • واجهات المستخدم (UI): تعلم كيفية تصميم واجهات المستخدم باستخدام XML.

واجهات المستخدم (UI)

واجهات المستخدم هي العناصر التفاعلية التي يتفاعل معها المستخدم. مثال على إنشاء زر والنقر عليه: 

          
import javax.swing.JButton;
import javax.swing.JFrame;
import javax.swing.JOptionPane;

public class Main {
    public static void main(String[] args) {
        JFrame frame = new JFrame("مثال على واجهات المستخدم");
        JButton button = new JButton("انقر هنا");

        button.addActionListener(e -> JOptionPane.showMessageDialog(null, "تم النقر على الزر!"));

        frame.add(button);
        frame.setSize(300, 200);
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setVisible(true);
    }
}

التفاعل مع قواعد البيانات (Database Interaction)

يمكن التعامل مع قواعد البيانات في جافا باستخدام JDBC. مثال على الاتصال بقاعدة بيانات MySQL واسترجاع البيانات: 

          
import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.ResultSet;
import java.sql.Statement;

public class DatabaseExample {
    public static void main(String[] args) {
        String url = "jdbc:mysql://localhost:3306/mydatabase";
        String user = "username";
        String password = "password";

        try {
            Connection conn = DriverManager.getConnection(url, user, password);
            Statement stmt = conn.createStatement();
            ResultSet rs = stmt.executeQuery("SELECT * FROM students");

            while (rs.next()) {
                System.out.println("الاسم: " + rs.getString("name") + ", العمر: " + rs.getInt("age"));
            }

            rs.close();
            stmt.close();
            conn.close();
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}
