تعلم تصميم مواقع الويب

ابدأ رحلتك في تعلم تصميم مواقع الويب باستخدام HTML, CSS, و JavaScript!

HTML

HTML هي لغة ترميز تستخدم لإنشاء هيكل صفحات الويب. هي الأساس الذي تبني عليه المواقع. 

                
<!DOCTYPE html>
<html lang="ar" dir="rtl">
<head>
    <meta charset="UTF-8">
    <title>موقعي الأول</title>
</head>
<body>
    <h1>مرحبًا بالعالم!</h1>
    <p>هذا هو موقعي الأول.</p>
</body>
</html>

CSS

CSS هي لغة تنسيق تستخدم لتزيين وتصميم صفحات الويب. يمكنك استخدامها لتغيير الألوان، الخطوط، والتخطيط. 

                
body {
    background-color: #121212;
    color: #ffffff;
    font-family: 'Cairo', sans-serif;
}

h1 {
    color: #00ff88;
    text-align: center;
}

JavaScript

JavaScript هي لغة برمجة تستخدم لإضافة التفاعل إلى صفحات الويب. يمكنك استخدامها لإنشاء عناصر تفاعلية مثل القوائم المتحركة والنوافذ المنبثقة. 

                
// مثال بسيط على JavaScript
document.querySelector('h1').addEventListener('click', function() {
    alert('تم النقر على العنوان!');
});
